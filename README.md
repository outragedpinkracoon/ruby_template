# A little template

Features:
* Minitest with pretty formatting
* Rubocop with minitest helpers and default ignores
* Rakefile to run Rubocop and tests
* Sample structure lib/test directory

Steps to use:
1. ``` bundle install ```

Run tests
* ``` Rake test ```

Run rubocop
* ``` Rake rubocop ```

Run both
* ``` Rake tidy ```
